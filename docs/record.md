# Record

## CSS

1. `>`（子选择器）：表示选择直接子元素。它将匹配指定元素的直接子元素，而不会选择更深层次的子元素。
2. `~`（兄弟选择器）：表示选择同级元素。它将匹配位于指定元素之后的所有同级元素。
3. `+`（相邻兄弟选择器）：表示选择直接跟在指定元素后面的同级元素，且必须紧挨着它。
